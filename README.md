"# Book-Store-Management-System-group" 
📚 Book Store Management System

A web-based Book Store Management System built using PHP (MVC Architecture) and MySQL. This system provides role-based functionalities for Admin, Employee, and Customer, allowing efficient management of books, orders, users, and store operations.

🚀 Features:

🔑 Authentication
User Registration & Login
Role-based access control (Admin, Employee, Customer)
Secure session handling

👨‍💼 Admin Panel
Dashboard overview
Manage books (Add, Edit, Delete)
Manage employees
Monitor activities/logs

👨‍🔧 Employee Panel
View assigned tasks
Manage books
Schedule management

👤 Customer Panel
Browse available books
Add to cart
Place orders
Manage addresses
View order history

📦 Order Management
Create and track orders
Order history for customers

🧾 Profile Management
Update user information
Change password

🛠️ Technologies Used
Backend: PHP (Core PHP with MVC structure)
Frontend: HTML, CSS, JavaScript
Database: MySQL
Architecture: MVC (Model-View-Controller)
